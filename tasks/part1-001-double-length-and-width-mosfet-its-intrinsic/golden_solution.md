# Golden Solution - part1-001-double-length-and-width-mosfet-its-intrinsic

The intrinsic gain is `Av,int = gm ro`.

If both `W` and `L` are doubled while the overdrive voltage is kept constant, `W/L` is unchanged, so `gm` is approximately unchanged. The drain current is also approximately unchanged, while `ro = 1/(lambda ID)` increases because `lambda` decreases as `L` increases. Thus `ro` approximately doubles, and the intrinsic gain approximately doubles.

This conclusion depends on the bias condition. If a different quantity is held fixed, the scaling can differ.
