# Golden Solution - part1-012-source-input-resistive-feedback

Figure 9 contains negative feedback, but the source of `M1` is not grounded. The input `Vin` is applied to the source of the NMOS, while the gate voltage `VF` is generated by the `R1`-`R2` divider from `Vout`.

The drain/output node is loaded by `RD`. A change in output voltage is divided down to the gate, so the circuit senses `Vout` and feeds a fraction back to the controlling terminal of `M1`. With the shown polarity, this is a negative-feedback structure.

It should therefore be analyzed by writing KCL at the source/input node and the output node, with `vF = vout R2/(R1 + R2)` in small signal. Treating the source as grounded turns the circuit into a different common-source amplifier and misses the actual input connection.
