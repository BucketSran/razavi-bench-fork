# Golden Solution - part1-024-many-poles-have

The circuit has two independent poles in the shown small-signal model.

One pole is associated with the input/source-side node containing `CGS` and related capacitances. The other is associated with the output node containing the drain-side capacitances and the Miller-coupled `CGD`. `CGS2` should not be counted as an output-node capacitance if the gate of `M2` is ac grounded; its terminal connection determines which node it loads.
