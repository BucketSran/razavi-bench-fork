# Golden Solution - part1-025-series-nmos-effective-long-channel

Figure 19 is two series NMOS devices sharing the same gate signal, with a resistive load at the top. If `M1` and `M2` are identical and biased together, the pair behaves like a single transistor with approximately twice the channel length.

This is not a differential half-circuit, not a gain-boosted cascode, and not a diode-connected load. The topology is useful when a longer effective channel is desired, for example to improve output resistance or reduce short-channel effects.
